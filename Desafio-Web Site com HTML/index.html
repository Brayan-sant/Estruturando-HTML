Clínica Geral é muito importante, afinal na maioria das vezes em que se vai em uma consulta inicial é o clínico geral quem recebe primeiro o paciente. Além disso, pode-se diagnosticar e tratar mais de 80% dos problemas de saúde quando se vai a uma clínica geral segundo a Sociedade Brasileira de Clínica Médica.

Isso ocorre, pois, o médico clínico geral é o primeiro profissional que as pessoas consultam quando estão com algum problema e não sabem para qual profissional especializado devem ir, logo o clínico geral deve ter conhecimento para não encaminhar problemas simples para outras especialidades, assim evitam-se exames desnecessários, mas caso precise é ele o responsável pelo encaminhamento para outras especialidades.
